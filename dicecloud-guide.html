Dicecloud Link: https://dicecloud.com/ Dicecloud is an online character sheet for DnD 5e. Like Gsheet, it allows for easy browser use (Works amazing in both mobile and PC), real time changes and even multiple people with edit permissions for the same sheet. It's based on the idea to track where each number comes from, and allow for easily changes on the fly. Let's look at a hypothetical example. You need to swim through a sunken section of dungeon to fetch the quest's Thing. You'll need to take off your magical Plate Armor of +1 Constitution to swim without sinking, of course. Taking it off will change your armor class, your speed and your constitution, which in turn changes your hit points and your constitution saving throw. Working out all those changes in the middle of a game will drag the game to a halt. Fortunately you have a digital character sheet, so it's a matter of dragging your Plate Armor +1 Con from your "equipment" box to your "backpack" box and you're done. Your hitpoints change correctly, your saving throws are up to date, your armor class goes back to reflecting the fact that you have natural armor from being a dragonborn. Your character sheet keeps up and you ultimately get more time to play the game. Huzzah! And all of these changes are synced with Avrae by using the !update command!
Creating a sheet After signing in to Dicecloud, first select the "Characters" tab on the left menu, then, click on the "+" button at the bottom right corner.
Then, from the popup menu, select "New Character"
A new box asking for some basic info about the character, after filling those fields, click on the "Add" button and create the new sheet.
In this example, we will be making the sheet for #tsunade-senju-the-doctor , so she will be a variant human female.
We will be sent to the basic empty sheet. The first step will be filling up the Race info, adding bonus stats, proficiencies, features, etc. Filling up the Race: First, we need to select the "Journal" tab where we will add(and/or modify) the info about the race and class(Or classes)
Once there, we will see the Race that was inputted at the moment of the sheet creation. We need to click on the Race to open up the editor.
A popup window will appear showing the info about the race, right now we will have nothing on it, so we need to enable the editor to start adding things. For this, we click on the top right corner of the popup.
Once enabled, we will be able to edit the Race name and add the extra bits needed. We will start with the basic stats like Speed and abilities scores. For this, we need to click on "Add Effect" to add each one of them. (edited)
A secondary popup will appear, showing us all the different options we can add as an effect for the race. For speed, we will select "Stat"
A secondary menu will appear under the "Stats" showing us all the different options. So we will chose "Speed"
Then, we will scroll up a little and on the right sub-menu, we will select "Base Value" and in the value input, we will add the Race base speed. In this case is 30. Note: In the value section we will only add the number in feets, no need to add the "ft" after the number.
Once everything is set up properly, we will click the back arrow at the top left corner to save the changes.
Now we will see that our race not only has the name but now it shows the speed too.
Next, we need to add the ability scores. Variant human gives one point in two different stats. For her we will add one point in Wisdom and another point in Strength. For each point, we need to add an extra effect. Once we click on "Add effect", we will select "Ability scores"
Then we will select the Stat (In this case Strength), on the sub menu at the right side, we will select "Add" since these are extra ability scores, and then we will add the value on the value input. At last we will save by clicking on the arrow at the top left corner, just like we did with speed.
Now we will see our next effect added to the Race.
Next, we will repeat the process and add the extra wisdom point.
Once that's done, we will have all something like this:
Next we need to add the extra skill proficiency and languages. For that we will click on "Add Proficiency" A new area will appear where we can select the proficiency.
First we will select, "Skill" and then we will select which one of the skills:
Next, we need to add the languages. For that, we will also add another proficiency, this time, in the "Stat Group" we will select "Language" and in the "Name" input, we will write the language we are adding.
In the end we will have something like this, and to save it, we will click on the checkmark at the top right corner of the popup.

Adding a class: Now, to add a class, we need to go to the journal tab and click on the "+" button:
We will be shown another very similar popup like the ones before, only this one has a field for the level of the class:
For our example, we will make our character a lvl 10 cleric:
Next we will start adding the effects, such as hit dice and hit points. Here we will have two options. Add the number manually and edit each time we level up, or use the the Dicecloud variables to have the sheet change the number automatically depending on our class level.
In this case, the variable is "Clericlevel", this works for any class we add, it doens't matter if it's a homebrew one. The variable always being "Classlevel" For example: Warlock -> Warlocklevel Druid -> Druidlevel Maid -> Maidlevel Overlord -> Overlordlevel
The same variable can be used to input the complete HP formula so the Sheet will automatically add the HP every time we level up. Note: There is no need to add the Constitution modifier since the sheet will automatically add that to the final HP value.
